## Deep Learning Models for Classification of RNA-seq Data: Balancing Performance and Computational Cost
Rhys Lossman  
MATH 5310  
April 23, 2026  


**Abstract** 

In bioinformatics literature there are many different deep 
learning models that achieve extremely high accuracy (greater than 95\%) for 
the task of classifying disease type or subtype based on RNA-seq data. To determine whether model 
architecture choice meaningfully impacts this performance accuracy,  the present study aims to compare 
basic versions of three typical classes of models used including feed-forward neural networks, 
convolutional neural networks, and recurrent neural networks and evaluate their performance on classifying 
TCGA-BRCA subtype. The study also considers the computational cost of training these models, to determine 
which class achieves the best balance between these two metrics. Additionally, models are trained on two 
different forms of dimensionally reduced data (through principal component analysis and with an autoencoder) 
to compare the efficacy of those two forms of data compression. The experimental findings suggest that FNNs 
outperform the other two classes when trained on the same data and that autoencoder embedding does not 
increase model performance. Future work should continue to explore how to assess computational complexity 
and model performance at the same time while also considering more nuanced variations of the aforementioned 
model architectures.

The files in this github are scripts which
* query and download TCGA-BRCA using `R` package `TCGAbiolinks`
* preprocess the data, dimensionally reduce it, and save it locally to feather files
* build and train 120 FNNs, CNNs, and RNNs to classify molecular subtype based on compressed input data
* perform hypothesis tests on the performance results from those models

The results shown here are generated in full thanks to data generated by the TCGA Research Network 
<https://www.cancer.gov/tcga>.
